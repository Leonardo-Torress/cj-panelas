# 🍳 CJ Panelas - Landing Page

Este projeto é uma **landing page de alta conversão** desenvolvida para a venda de **conjuntos de panelas antiaderentes**, com foco em design moderno, persuasão e experiência do usuário.

---

## 🚀 Sobre o Projeto

A CJ Panelas é uma página criada com o objetivo de apresentar e vender conjuntos de panelas de forma clara, atrativa e confiável, utilizando técnicas de marketing digital como:

* Prova social (depoimentos de clientes)
* Destaque de benefícios do produto
* Design responsivo
* Estrutura otimizada para conversão

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📂 Estrutura do Projeto

```
cj-panelas/

```
